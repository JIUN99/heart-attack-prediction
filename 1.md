# Heart Attack Prediction — GenAI Chatbot

A conversational AI chatbot that assesses cardiac risk using a **Transformer + Neural Network** model trained on hybrid tabular + NLP features.

## Architecture

```
User → Claude Chatbot (Anthropic API)
           ↓  collects health info conversationally
       Transformer + NN model
       (tabular features + sentence embeddings + sentiment)
           ↓
       Risk prediction (High / Low) + probability
           ↓
       Claude explains results & gives lifestyle advice
```

## Model Pipeline

| Step | Component |
|------|-----------|
| Text generation | Rule-based (train-time) / Mistral-7B-Instruct (optional) |
| Sentence embeddings | `all-MiniLM-L6-v2` (384-dim) |
| Sentiment | `distilbert-base-uncased-finetuned-sst-2-english` |
| Classifier | Dense 256 → BatchNorm → Dropout → 128 → 64 → 1 (sigmoid) |
| Features | Age, Gender, BMI, BP, Cholesterol, Triglycerides, Fasting Sugar, CRP, Homocysteine, Sleep, Smoking, Diabetes, Family History, Exercise, Stress, Alcohol, + 384 embeddings + Sentiment |

## Local Setup

```bash
git clone https://github.com/JIUN99/heart-attack-prediction.git
cd heart-attack-prediction
pip install -r requirements.txt

# Train and save model artifacts to ./model/
python train_model.py

# Set your Anthropic API key
export ANTHROPIC_API_KEY=sk-ant-...

# Run the chatbot locally
python app.py
# → open http://localhost:5000
```

## Deploy to Render

1. Push this repo to GitHub.
2. Go to [render.com](https://render.com) → **New Web Service** → connect your GitHub repo.
3. Render auto-detects `render.yaml`. Set the `ANTHROPIC_API_KEY` environment variable in the dashboard.
4. Deploy — Render will run `python train_model.py` at build time, then start `gunicorn app:app`.

> ⚠️ Use the **Standard** plan (≥ 2 GB RAM) for the embedding models.
